# 关系上的标签
在关系之间使用标签来说明时, 使用 :后接 标签文字。

对元素的说明，你可以在每一边使用 "" 来说明.

``` puml
@startuml

类01 "1" *-- "many" 类02 : 包含

类03 o-- 类04 : 聚合

类05 --> "1" 类06

@enduml
```

在标签的开始或结束位置添加< 或 >以表明是哪个对象作用到哪个对象上。

``` puml
@startuml
class 汽车

发动机 - 汽车 : 驱动 >
汽车 *- 轮子 : 拥有 4 >
汽车 -- 人 : < 所属

@enduml
```

### 练习

动物 “1”<--“2” 猫咪:继承

class 发电机
class 电路
class 电灯
发电机 --> 电路:发电 >
电路 --> 电灯:输电 >
电灯 --> 黑夜:照亮 >
